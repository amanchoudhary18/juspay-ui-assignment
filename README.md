- **@tailwindcss/vite:** `^4.1.17`

## Features

- Responsive Design
- Microinteractions
- Theme Toggle (Dark/Light)
- React Context for State Management
- Custom Hooks

## Installation & Setup

### Steps to Run

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
